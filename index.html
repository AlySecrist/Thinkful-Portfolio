Harry Potter Quiz

Live App
GitHub Repo

The Harry Potter Quiz is a short, 5 question trivia quiz based on the Harry Potter series of books by J. K. Rowling.It was built using Javascript and JQuery, HTML, and CSS for anyone interested in testing their basic Harry Potter knowledge, or a quick laugh.

This project taught me how to personalize UX so that the user feels more welcomed and involved by asking for the user's name and returning that input throughout the quiz by calling the user by their provided name.
